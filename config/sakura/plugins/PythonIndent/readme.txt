Pythonスマートインデントプラグイン

＜特徴＞
・ifやfor, whileのような構文に反応して次の行を字下げします。

＜使用方法＞
・プラグインに対応している開発版sakuraW.exeと同じフォルダの中に
　plugins というフォルダを作り、その中にPythonIndentフォルダを配置
　してください。
・サクラエディタを起動し、Pythonのタイプ別設定から以下を設定します。
　（Pythonのタイプ別設定は自分で作ってください）
　「スクリーン」タブ
　　　「インデント」ー「スマートインデント」の選択肢からPythonを選択。
　　　「インデント」―「自動インデント」にチェックを入れる。【必須】
　　　「インデント」―「SPACEの挿入」にチェックを入れる。【必須】
　　　「レイアウト」―「TAB幅」を希望の幅に合わせる。
・これで準備完了です。
